Employee Management System

Employee Management System is a web-based application designed to manage employees within an organization.

It utilizes Spring Boot for the backend, Angular for the frontend, and SQL Server for storing employee-related data.

Key Features:

  • User Login and Role-based Access:
    • User: Can view the employee list and see individual details.
    • Admin: Can delete, update, and add new employees.
  • Database Management: The system connects to a SQL Server database for storing employee data.

Flavor Heaven

Flavor Heaven is a web-based recipe discovery and management application.

It utilizes HTML, CSS, JavaScript, MSSQL, Spring Boot, Angular for an interactive and dynamic user experience.

Key Features:

  • Dish of the Day: Showcases a randomly selected featured recipe on the homepage daily.
  • Advanced Search Functionality: Users can find recipes by searching for dish names or ingredients.
  • Recipe Categorization:
    • Dish types: Pizza, Burger, Dosa, Ice Cream, Biryani, Samosa.
    • Cuisine-based classification: Indian, Chinese, Italian, Mexican, etc.
    • Meal-based classification: Breakfast, Main Course, Snacks, Beverages.
